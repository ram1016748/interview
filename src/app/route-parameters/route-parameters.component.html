ROUTE PARAMETERS

  • User click on any of the course it should navigate to course-details component by sending that course particular ID as a parameter
  • I have created a course list array in course list component and displayed those courses into view.
  • As discussed before when the user clicks on any course to navigate the course-details component, so first need to generate the course-details component.
  • to navigate between the component need to create routings.
  • In the course list add the click event and should construct the route URL on that event method, so we need to use the router service.
  • To use the Router service, we should Inject the Router service in the course-list Constructor()
  • Navigating method accepted array called 'Link Parameters Array'(navigate([ ])), It provides all the information to the angular to construct the URL


  • Where to read the ID details in that component(course-details) we should Inject the 'ActivatedRoute'

    • Note:

  • paramMap is a API, which helps to get the parameters from the URL.
