TWO-WAY DATA BINDING:

  • Two-way data binding refers to sharing data between a component class and its template.
  • If you change data in one place, it will automatically reflate at the other end.
  • ngModel directive is used to achieve two-way data binding.
  • Must import FormsModule from '@angular/forms'
  • [ ] - Property Binding
  • ( ) - Event Binding
  • Two-way Data binding = Property Binding + Event Binding
