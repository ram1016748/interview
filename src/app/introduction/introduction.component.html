INTRODUCTION:

  • Angular is an open source JavaScript binding frameWork to build the web Applications (Clint Side)
  • Angular binds the Model with View Architecture
  • Angular helps to implement MVVM (Model View View Model) pattern
  • Angular is used to develop a SPAs(Single Page Applications)
  • Angular helps to load parts of web pages asynchronously rather then load entire page.
  • By using components future, we have a lot of re-usable code
  • In angular it is possible to do Unit test and easily maintain the code
  • Angular is the product of Google team and makes us of Type Script language from Microsoft.

TypeScript:

  • Type Script is a Open-Source pure Object-Oriented Programming Language
  • It is a strongly typed superset of Java Script, which compiles to plain Java Script
  • It is developed and Maintained by Microsoft under Apache License.
  • It is not directly run on the browser. It needs a compiler to compile and generate a java Script File.(uses V* Engine)
  • Type Script source files is in ".ts" extension
  • Finally: Type Script = Java Script + Some additional features

TypeScript Advantages:

  • TypeScript always highlights errors at compilation time, during the time of development, whereas Java Script points our errors at run time.
  • Type Script supports strongly typed or static typing, whereas this is not in Java Script.
  • Type Script runs on any browsers.
  • it has a namespace concept by defining a module.

TypeScript Disadvantages:

  • Type Script takes a long time to compile the code.(Once application is compiled next time it compile fastest )
  • Type Script does not support Abstract Classes.
  • To run Type Script application in browser, a compilation step is required to transform type script to Java Script.
